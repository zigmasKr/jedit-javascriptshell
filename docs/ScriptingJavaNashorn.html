Scripting Java with Nashorn

Nashorn is JavaScript implemented on Java. There are many books and tutorials on JavaScript. On the other hand, some people say that Java specific features of Nashorn that allow scripting Java objects are not well described and covered with tutorials. Below is a short list of sources which are usefull in studying Nashorn. It is not complete, but includes official Oracle pages as well as most informative tutorials.

  • https://docs.oracle.com/javase/8/docs/technotes/guides/scripting/prog_guide/api.html
  • http://www.oracle.com/technetwork/articles/java/jf14-nashorn-2126515.html
  • https://docs.oracle.com/javase/9/nashorn/toc.htm
  • https://wiki.openjdk.java.net/display/Nashorn/Nashorn+extensions
  • https://wiki.openjdk.java.net/display/Nashorn/Rhino+Migration+Guide
  • http://winterbe.com/posts/2014/04/05/java8-nashorn-tutorial/
  • The book "Scripting in Java" by Kishori Sharan, Apress, 2014.
